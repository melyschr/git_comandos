GIT - Principais Comandos


Afinal, o que é o git e para que ele serve?

Git é um controle de versionamento de codigos e projetos, e serve para controlar as auterações feira codigo por versões, podendo armazena-las alem de outros recursos, como voltar a um estado ou versão anterior do codigo, sendo uma ferramenta muito importante para qualquer desenvolvedor ou programador.


1 - Alguns comandos basicos

    Navegando nas pastas do seu computador no git bash (Windows)

  • cd C:/Users/ (entrar no diretorio escolhido)
    cd ../ (voltar um diretorio)

    • Criar pastas

  • mkdir nomepasta

    • Listar arquivos

  • ls
    dir
    tree /f

2 - Configurando o Git

Alguns primeiros comandos que fazemos ao baixar o git, é configura-lo com nossas informações:

    Adicionando seu nome de usuario

  • git config --global user.name "seu nome"

    • Adicionando email da sua conta

  • git config --global user.email "seuemail@email.com" (email da sua conta do github)

    • Adicionando seu editor de código/texto

  • git config --global core.editor sub (no meu caso sub = sublime text)

3 - Criando Repositório

    Iniciar repositorio, na pasta ja escolhida para o seu repositório

  • git init

    • Adicionar arquivos ao monitoramento git

  • git add nome.extensão;
    git add -A (todos)

    • Enviar e salvar alterações dos arquivos adicionados no git, em versões

  • git commit -m "Comentario do commit feito"
